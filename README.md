## 2. **Short Description/Purpose**
This project analyzes a year's worth of pizza sales data (Jan-Dec 2015) to uncover business insights, customer preferences, and sales trends. The goal is to help pizza restaurant owners understand:
- Sales performance across different pizza categories and sizes
- Best and worst-selling pizzas
- Daily and monthly order trends
- Customer purchasing patterns

## 3. **Tech Stack**
- **Database**: PostgreSQL (via pgAdmin)
- **Data Analysis**: SQL Queries
- **Data Visualization**: Microsoft Power BI
- **Version Control**: GitHub
- **Data Source**: CSV file (Kaggle)

## 4. **Data Source**
- **Platform**: Kaggle
- **File Type**: CSV
- **Records**: 48,620 orders
- **Time Period**: January 2015 - December 2015
- **Columns Includes**: 
  - `pizza_id`, `order_id`, `pizza_name_id`, `quantity`
  - `order_date`, `order_time`, `unit_price`, `total_price`
  - `pizza_size`, `pizza_category`, `pizza_ingredients`, `pizza_name`
